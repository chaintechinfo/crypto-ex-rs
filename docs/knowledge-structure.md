# Rust 知识结构

## 目标

1. 掌握 Rust 基本语法
2. 掌握所有权、借用、生命周期等
3. 掌握复合数据结构：结构体、元祖、枚举
4. 掌握 Trait 和泛型

基础必备

0. 基本语法和基础数据类型
1. 所有权：ownership, borrowing, copy, move
2. 字符串
   1. String, &str, &'static str
   3. Vec<u8>, &[u8], &[u8; N]
   4. &str, &[u8]
   5. OsStr, OsString, Path, PathBuf, CStr, CString
3. 复合类型：结构体、元祖
4. Trait
   1. Debug Trait
   2. Display Trait
   3. FromStr Trait
   4. Copy Trait
5. 错误处理
